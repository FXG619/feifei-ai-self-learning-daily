# 审核 Checklist

飞飞审核草稿时使用本清单。任何 AI 生成内容默认是 `draft`，通过人工审核后才可改为 `published`。

## 内容审核

- [ ] 有没有制造家长焦虑？
- [ ] 有没有承诺具体提分？
- [ ] 有没有让 AI 替孩子做作业？
- [ ] 有没有鼓励孩子主动思考？
- [ ] 有没有一个具体学习动作？
- [ ] 有没有“今天只做一件事”？
- [ ] 有没有“家长今天可以问”的一句话？
- [ ] 有没有“不要误用 AI”的提醒？
- [ ] 家长是否能看懂？
- [ ] 孩子是否能照着做？
- [ ] 语气是否温和、清楚、不营销？

## 技术审核

- [ ] `status` 是否为 `draft` 或 `published`？
- [ ] `draft` 是否不会公开展示？
- [ ] RSS 是否排除了 `draft`？
- [ ] frontmatter 是否完整？
- [ ] AI prompt 是否存在？
- [ ] 风险提醒是否存在？
- [ ] 是否没有提交 API Key 或 secrets？
