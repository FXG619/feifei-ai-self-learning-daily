# MVP 验收标准

1. `npm install` 成功。
2. `npm run build` 成功。
3. `npm run validate:content` 成功。
4. 首页只展示最新 `published` 日报。
5. `/daily` 只展示 `published` 日报。
6. `/rss.xml` 只包含 `published` 日报。
7. `draft` 内容不会公开展示。
8. `npm run generate:draft` 可以生成新 `draft`。
9. 至少有 1 篇 `published` 样例日报。
10. 至少有 2 篇 `draft` 样例日报。
11. 文章页有复制 AI 提示词按钮。
12. README 解释如何审核和发布。
13. 文案没有承诺提分。
14. 文案没有制造家长焦虑。
15. 文案没有鼓励孩子直接向 AI 要答案。
16. 首页首屏突出“今天只做一件事”。
